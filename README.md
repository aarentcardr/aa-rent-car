# 🚗 AA Rent Car — Sitio Web Oficial

Sitio web institucional de **AA Rent Car**, empresa de renta de vehículos ubicada en **La Romana, República Dominicana**.

---

## 📋 Descripción

Página web moderna y responsiva que presenta la flota de vehículos disponibles, los servicios ofrecidos y los canales de contacto de AA Rent Car. Diseñada para facilitar la reserva directa vía WhatsApp.

---

## ✨ Características

- Diseño 100% responsivo (móvil, tablet y escritorio)
- Filtro interactivo de vehículos por categoría
- Integración directa con WhatsApp para reservas
- Sección de flota con precios actualizados
- Formulario de contacto con redirección a WhatsApp
- Animaciones suaves al hacer scroll
